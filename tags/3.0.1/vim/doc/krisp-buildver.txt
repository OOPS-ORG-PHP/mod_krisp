*krisp_buildver* -- Returns version of mod_krisp

  (string) krisp_buildver (void)~

|krisp_buildver| 함수는 mod_krisp 확장의 버전을 반환한다.

다음은 |krisp_buildver| 함수의 출력 결과를 보여준다.

<?php >
  echo "The version of mod_krisp extension is ";
  echo krisp_buildver () . ".\n";
?>

위의 예제의 결과는 다음과 같다:

  The version of mod_krisp extension is 2.1.2.
